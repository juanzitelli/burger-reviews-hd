## Run Locally

Clone the project

```zsh
  git clone git@github.com:juanzitelli/burger-reviews-hd.git
```

Go to the project directory

```zsh
  cd burger-reviews-hd
```

Install dependencies

```zsh
  yarn
```

Start the server

```zsh
  yarn dev
```
